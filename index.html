The NeuroDISK Project


Brain image and genetic data can be significant for researchers to understand brain structure, functions and disease. Though data generation is continuous, analysis of these data can only perform once and mainly based on recently generated data. With new data coming in, the original hypothesis may need to be revised.

Our NeuroDISK project aims to make the analysis automated. In NeuroDISK, we develop an ontology Organic Data Science framework (ODS) built on wiki to do data management, and implement meta-regression workflows in WINGS to do data analysis. Then, DISK allows us to decide the variables of interest to particular data fields. It will then trigger a line-of-inquiry (LOI) to propose, test or update a hypothesis automatically. Our template is the effect size of rs1080066 of surface area on Precentral is associated with the mean age of the cohort.
